PyMunin System Resources Plugin

Multigraph Munin Plugin for monitoring System Resources

Load Average, CPU and Memory Utilization, Processes, Forks, Interrupts, Context Switches, Paging and Swapping etc.

System Resources Plugin - sysstats

The multigraph plugin implements the following graphs:

  • sys_loadavg
  • sys_cpu_util
  • sys_memory_util
  • sys_memory_avail
  • sys_processes
  • sys_forks
  • sys_intr_ctxt
  • sys_vm_paging
  • sys_vm_swapping
